NBAZone Frontend:

NBAZone is a full-stack sports analytics platform for exploring NBA player performance data.
This repository contains the frontend application built with React.


Features: (In Progress)

Player search interface
Player profile pages
Season statistics display
Integration with backend API


Tech Stack:

React
JavaScript


Backend API:

This frontend connects to:
https://github.com/quiszioce/NBAZone_Backend


Running the Project:

Prerequisites:
Node.js
npm

Steps:
Clone the repo
git clone https://github.com/quiszioce/NBAZone_Frontend.git

Navigate into folder
cd nbazone-frontend

Install dependencies
npm install

Start the app
npm start
